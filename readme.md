# Todo List - React Native & TypeScript

Proyecto desarrollado para la asignatura de [Nombre de la Materia].

## 🚀 Características
- Gestión de tareas (CRUD básico).
- Tipado estricto con TypeScript.
- Interfaz reactiva con Hooks.

## 🛠️ Configuración y Ejecución

1. **Clonar el repositorio:**
   `git clone https://github.com/tu-usuario/todo-list-rn.git`

2. **Instalar dependencias:**
   `npm install` o `yarn install`

3. **Iniciar el servidor de desarrollo:**
   `npx expo start`

4. **Visualización:**
   - Escanea el código QR con la app **Expo Go** en Android/iOS.
   - O presiona `w` para abrir en el navegador.

## 📁 Estructura del Proyecto
- `/src/components`: Componentes visuales.
- `/src/types`: Interfaces de TypeScript.
- `App.tsx`: Punto de entrada y lógica de estado.
